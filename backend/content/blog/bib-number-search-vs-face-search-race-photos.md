# Bib Number Search vs Face Search: Mana Lebih Sesuai Untuk Cari Race Photos?

Dua cara paling biasa untuk mencari gambar race ialah **bib number search** dan **face-search**. Kedua-duanya berguna, tetapi situasinya berbeza.

## Apa itu bib number search?

Bib number search mencari gambar berdasarkan nombor peserta yang tertera pada bib. Jika nombor bib jelas dalam gambar, sistem boleh memadankan gambar dengan nombor tersebut.

Kelebihan:

- cepat untuk gambar front-facing
- sesuai untuk finish line dan checkpoint
- tidak perlu upload selfie

Kekurangan:

- gagal jika bib tertutup tangan, hydration vest, atau race belt
- nombor kabur jika motion blur
- kurang sesuai untuk angle tepi atau belakang

## Apa itu face-search?

Face-search mencari gambar berdasarkan wajah runner. Biasanya runner upload selfie, kemudian sistem mencari wajah yang sepadan dalam gallery.

Kelebihan:

- boleh jumpa gambar walaupun bib tidak kelihatan
- berguna untuk candid route photos
- sesuai untuk event besar dengan ribuan gambar

Kekurangan:

- selfie perlu jelas
- sunglasses, cap, atau lighting ekstrem boleh mengurangkan ketepatan
- tidak semua gallery menyediakan fungsi ini

## Bila patut guna bib search?

Guna bib search dahulu jika anda tahu nombor bib dan event mempunyai gambar finish line yang jelas. Ini biasanya cara paling pantas.

## Bila patut guna face-search?

Guna face-search jika bib tidak jumpa, anda memakai race belt, atau mahu cari semua candid sepanjang route.

## Cadangan terbaik

Gunakan kedua-duanya. Mulakan dengan bib number search, kemudian gunakan face-search untuk cari gambar tambahan yang mungkin terlepas.
